This is a minecraft afk bot which joins your minecraft server and keeps the server alive.

Fork this  repository.

replace your ip and port(if your server uses a custom port) in the index.js file.

replace username with your desider user name in the index.js file.

visit render.com and sign in with your git hub account.

once signed in , click on deploy a web service.

click on afk-minecraft repository and choose the free tier.

once the service is live and the bot joins your server visit cron-job.org.

this will keep your free render instance alive.

copy the link below your account name and paste it in cron-job.org

now the bot should join your server and keep the server alive 24/7.
